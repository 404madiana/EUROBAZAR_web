# Un exemplaire de notice

Voici un aperçu d'un fichier nommé "exemple.md".
Le principe est d'afficher le contenu d'un fichier MarkDown via ReactMarkdown.

## Pourquoi faire ??

Le site va utiliser des fichiers markdown pour rédiger des remarques d'utilité public concernant plusieurs aspects du magasin EUROBAZAR. 
Ce fichier est un test pour l'implémentation de l'affichage.

## Pourquoi Markdown ?

Markdown est facile et pratique. jpense c'est une raison suffisante..

La fonctionnalité d'affichge Markdown est inspiré des README modifiables sur GitHub.

Tout ce qui est affiché ici est pratiquement le même contenu que sur le fichier originel. La seule différence est que le fichier originel comporte des caractères de ponctuation pour enrichir le texte ici (titres, **gras**/*italique*/`code`...).
