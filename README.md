# Site web pour EUROBAZAR

Ceci est un projet web de vitrine numérique pour le magasin EUROBAZAR.<br>
Le projet a pour but d'augmenter la visibilité numérique du magasin et d'ancrer l'enseigne sur Internet.

## Magasin EUROBAZAR

Eurobazar est une enseigne se situant à `36 Avenue de Fontainebleau, 94270 Le Kremlin-Bicêtre`.<br>
Il est un magasin composé de deux étages et dirigé par Said DOUICH.
Sous un atmosphère oriental, ce magasin possède un éventail large de produits alimentaires tels que des fruits secs, noix, épices, bonbons, thés et cafés, pâtes...<br>On y trouve également des articles ménagères, de décoration, et d'ustensiles du quotidien. Vous y trouverez peut-être des articles obtenable nulle part ailleurs 👀
